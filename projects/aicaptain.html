AI Captain

VS Code Chat Participant mit Copilot + Perplexity Web-Research

Code ansehen

Über das Projekt

AI Captain ist eine VS Code Extension, die als Chat Participant in GitHub Copilot Chat integriert wird. Mit @aicaptain in Copilot Chat kombiniert sie die Coding-Power von Copilot mit Perplexity Web-Research — für fundierte Code-Reviews, Feature-Implementierungen und Recherche direkt in der IDE.

Das Tool vereint Code-Generierung, automatische Reviews und aktuelle Web-basierte Recherche in einem einzigen Workflow, optimiert für professionelle Softwareentwicklung.

Features

/code Command

Feature-Implementierung mit automatischem Code-Review-Loop. Bis zu 5 Iterationen für optimale Ergebnisse.

/review Command

Code-Review der aktuellen Datei oder Selektion mit Web-gestützten Best Practices.

/research Command

Tiefgreifende Web-Recherche zu beliebigen Themen direkt in VS Code mit Quellenangaben.

/plan Command

Detaillierte Implementierungspläne erstellen mit Kontext aus dem Codebase und Web-Recherche.

Multi-Model Architektur

Kombiniert Copilot (Coding) + Perplexity Sonar Pro (Research) für bestmögliche Ergebnisse.

Konfigurierbar

Wählbare Perplexity-Modelle (Sonar, Sonar Pro, Sonar Reasoning Pro) und anpassbare Iterationslimits.

Technologien

TypeScript 5.7 VS Code API GitHub Copilot Chat API Perplexity API esbuild Node.js 20
