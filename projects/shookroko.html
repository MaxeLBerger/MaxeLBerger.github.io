Shookroko

Browser-basiertes Survivors-Game mit 6 Umgebungen

Code ansehen

Über das Projekt

Shookroko ist ein browser-basiertes Survivors-Style Game mit einem niedlichen Amphibien-Charakter, der endlose Wellen von Feinden in verschiedenen aquatischen Umgebungen bekämpft. Das Spiel bietet tiefe Customization, prozedurales Audio und persistente Progression.

Gebaut mit React 19, TypeScript und HTML5 Canvas mit über 3.500 Zeilen Entwickler-Dokumentation — bereit für professionelle Weiterentwicklung.

Features

6 Umgebungen

Sumpf, Fluss, Korallenriff, Abgrund, Vulkan und Gefrorener See — jede mit einzigartiger Atmosphäre.

Progressives Wave-System

Kontinuierliches Multi-Directional Enemy Spawning mit skalierender Schwierigkeit.

Deep Customization

Skins, Hüte, Augenausdrücke, Emotes und Waffen-Skins für den Spielercharakter.

Einzigartige Feinde

6 umgebungsspezifische Feindvarianten mit speziellen Verhaltensweisen.

Prozedurales Audio

Echtzeit-synthetisierte Ambient-Sounds über die Web Audio API.

Persistente Progression

Unlocks und Customization werden via KV Storage dauerhaft gespeichert.

Technologien

React 19 TypeScript 5.7 Vite 7 HTML5 Canvas Web Audio API Tailwind CSS Shadcn UI
