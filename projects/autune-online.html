Über das Projekt

AuTune Online ist ein professionelles Audio-Visualisierungs-Tool, das MP3-Dateien analysiert und atemberaubende visuelle Effekte basierend auf Frequenzen und BPM erzeugt.

Features

MP3 Upload

Drag & Drop oder File-Dialog

Frequenzanalyse

Web Audio API Spektrum-Analyse

BPM-Erkennung

Automatische Tempo-Erkennung

Frequenz-Bars

Live Audio-Spektrum Visualisierung

Partikel-Effekte

Reaktiv auf Bass und BPM

Verschiedene Themes

Anpassbare Visualisierungen

Visualisierungs-Modi

Circles Mode

Eine dynamische Kreis-Visualisierung, bei der Frequenzen radial dargestellt werden. Die Größe und Intensität der Kreise reagiert direkt auf die Bass-Frequenzen und schafft so ein pulsierendes Erlebnis.

Grid Mode

Eine futuristische Gitter-Darstellung, die an Retro-Sci-Fi-Ästhetik erinnert. Das Gitter verformt sich im Takt der Musik und bietet eine räumliche Tiefe, die besonders bei elektronischer Musik zur Geltung kommt.

Spiral Mode

Hypnotische Spiralen, die sich im Rhythmus drehen und ausdehnen. Dieser Modus nutzt komplexe mathematische Funktionen, um Frequenzen in eine fließende, organische Bewegung zu übersetzen.

Anpassbare Farbschemata

Wähle aus verschiedenen Farbmodi wie "Rainbow", "Warm", "Cool" oder "Happy", um die Stimmung der Visualisierung an deine Musik anzupassen. Jedes Schema bietet eine einzigartige Farbpalette für ein individuelles Erlebnis.

Technologien

JavaScript Web Audio API Canvas API Node.js Express HTML5/CSS3

Wie es funktioniert

  1. Lade eine MP3-Datei hoch (Drag & Drop)
  2. Die App analysiert das Audio via Web Audio API
  3. BPM wird automatisch erkannt
  4. Frequenzen werden in Echtzeit visualisiert
  5. Partikel reagieren auf Bass-Frequenzen
  6. Genieße die synchronisierten visuellen Effekte!
