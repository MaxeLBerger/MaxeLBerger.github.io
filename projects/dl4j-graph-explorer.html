Über das Projekt

DL4J Graph Explorer ist ein read-only Visualisierungstool für DeepLearning4J (DL4J) Computational Graph Modelle. Importiere deine trainierten Modelle und erkunde ihre Architektur visuell mit interaktiven Graphen.

Das Tool wurde mit einem Local-First Ansatz entwickelt - keine externen Backends, alle Daten werden lokal im Browser verarbeitet und in localStorage gespeichert.

Features

Model Import

DL4J JSON oder ZIP Model-Exporte importieren

Layer Topology

Inbound/Outbound Connections inspizieren

Parameter Count

Per-Layer Parameter-Anzahl anzeigen

Weight Statistics

Histogramme, Mean und Std Dev erkunden

Graph View

Interaktive D3.js Netzwerk-Visualisierung

Table View

Tabellarische Weight-Übersicht

Design-Prinzipien

Local-First

  • Kein externes Backend
  • Daten bleiben lokal
  • localStorage Persistenz

Read-Only

  • Kein Training
  • Keine Mutation
  • Keine Inferenz

Fast Feedback

  • Schneller Import
  • Import Summary
  • Sofortige Visualisierung

Accessible UI

  • Keyboard-Friendly
  • Radix UI Primitives
  • Klare Navigation

Anleitung

  1. Model exportieren: Exportiere dein DL4J Model als JSON oder ZIP
  2. Datei importieren: Ziehe die Datei in den Graph Explorer
  3. Topologie erkunden: Erkunde die Layer-Topologie im Graph View
  4. Layer inspizieren: Klicke auf Layer für Details und Verbindungen
  5. Statistiken analysieren: Analysiere Weight-Statistiken und Histogramme
  6. Table View: Wechsle zur Table View für tabellarische Übersicht

Technologien

TypeScript React Vite D3.js Tailwind CSS Radix UI
